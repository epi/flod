    API documentation

    Module Description
    flod.adapter Adapters connecting stages with incompatible interfaces.
    flod.buffer Various buffer implementations.
    flod.common Commonly used stream sources, filters and sinks.
    flod.file Streams for reading and writing files.
    flod.meta Various metaprogramming helpers.
    flod.pipeline Pipeline composition.
    flod.range Convert pipelines to ranges (of chunks, text lines, etc.).
    flod.traits This module defines templates for determining characteristics of stream components at compile time.
    flod High-level interface for flod. Provides the most commonly used functions of the package.
